# Tanburi Küçük Artin – Makam / Terkib Analizi (Güncel v2)

## Bu sürümde düzeltmeler
- Karar/Merkez seçenekleri **tüm perde envanterinden** dinamik üretilir (3 seçenekle sınırlı değildir).
- Alt/Üst sınır ve Başlangıç civarı alanları **genişletildi** (dinamik).
- Asıl perdeler listesi **ASIL_MAP** (kilitli eşleme) üzerinden gelir.
- Nim perdeler listesi, veri setindeki diğer perdelerden otomatik çıkarılır.
- Perde sıralaması (Artin örnek seyri) **detay panelinde** gösterilir.

## 17'li sistem notu
- Dik Kürdi → Segâh
- Dik Acem → Evc
- Dik Acem-Aşiran → Irâk
- Şehnaz terkibinde dik acem yerine Evc kullanılır.
