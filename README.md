# tanburi-artin-nazariyesi-ezgi-cozumlemesi
Bu tahliller Tanburi Küçük Artin’in Musikî Edvârı adlı eserinde
makamların “bir mızrab” ile verilen melodik seyirlerine dayanır.
Eserde âgâz, merkez ve karar perdeleri kavramsal başlıklar halinde
tanımlanmadığından, bu alanlar yalnızca seyir dizilerinin
başlangıç–bitiş, tekrar ve yoğunluk özelliklerinden türetilmiştir.
Başka nazariyeci sistemleriyle karşılaştırma yapılmamıştır.
