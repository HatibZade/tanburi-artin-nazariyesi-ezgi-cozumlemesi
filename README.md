
# Tanburi Küçük Artin – Makam / Terkib Analizi

Bu sürümde:
- Perde sıralaması **detay panelinde** gösterilir
- Öğrenci / analist eserin:
  - Nazari seyir davranışlarını
  - Alt / üst alanlarını
  - Asıl ve nim perde kullanımını
  girebilir
- Makam sonuçları olasılık olarak sunulur

Sistem Tanburi Küçük Artin'in **17'li ses sistemi** esas alınarak hazırlanmıştır.
